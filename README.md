# TFG
El objetivo de este proyecto es desarrollar una aplicación que facilite la coordinación y gestión de grupos de trabajo en la UPO (equipo rectoral, equipos decanales, consejos de departamento, comisiones, etc.). 
Actualmente estas tareas —agenda reuniones documentos actas notificaciones etc— se gestionan con herramientas diferentes y no interconectadas. 
Este TFG busca unificar en una sola plataforma web la planificación de agendas convocatorias seguimiento de reuniones almacenamiento y acceso seguro a documentación así como la emisión de avisos y recordatorios automáticos. 
La aplicación deberá hacer uso delas aplicaciones corporativas necesarias para su despliegue real por ejemplo autenticación con SSO (adAS) o almacenamiento en la nueve (MS OneDrive).
